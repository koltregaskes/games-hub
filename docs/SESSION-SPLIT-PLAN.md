# Session Split Plan

The studio is now big enough that the next serious work should split into title-specific sessions.

## Recommended dedicated sessions

### Session 1: Mandate 2029

- Goal: push toward a Democracy-style full political sim
- Focus: events, policy systems, voters, cabinet, balancing

### Session 2: Civicrise

- Goal: push toward a modern SimCity / Cities-style experience
- Focus: zoning, budgets, transport, progression, city readability

### Session 3: Neon District

- Goal: turn the district slice into a proper cyberpunk action vertical slice
- Focus: combat, cover, mission design, atmosphere

### Session 4: Starfall Protocol

- Goal: turn the orbital dock demo into a real third-person game foundation
- Focus: traversal, interaction, stealth/hacking fantasy, mission framing

### Session 5: Turbo Vector

- Goal: build the `Super Cars` answer
- Focus: garage economy, rivals, one championship loop, richer track set

### Session 6: Redline Horizon

- Goal: build the `Lotus Esprit Turbo Challenge` answer with `OutRun` fantasy
- Focus: stages, scenery, route splits, music identity

### Session 7: Swarmbreaker

- Goal: finish the arcade lane to a tighter release standard
- Focus: wave design, enemy set, presentation polish

## Why split

- each game now has its own identity
- each one wants different mechanics and references
- title-specific sessions will keep context cleaner and progress faster
