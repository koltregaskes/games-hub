# Game Plans

This is the current per-game plan sheet for the active slate.

## Swarmbreaker

- Current state: playable desktop arcade game
- Immediate target: stronger enemy variety, cleaner wave escalation, sharper audio/visual impact
- Near-term milestone: one polished arcade release candidate
- Session focus: enemy behaviours, score systems, presentation polish

## Mandate 2029

- Current state: strong political strategy prototype
- Immediate target: deeper policy interactions, more opposition/cabinet play, better balance
- Near-term milestone: a fuller campaign with stronger realism and replayability
- Session focus: mechanics depth, content expansion, balancing

## Civicrise

- Current state: verified playable city-building simulation
- Immediate target: stronger zoning/economy loops, transport clarity, better long-session progression
- Near-term milestone: a convincing small-city-to-metropolis growth arc
- Session focus: simulation depth, interface clarity, progression tuning

## Neon District

- Current state: environment-first cyberpunk district slice with optional combat
- Immediate target: cover logic, denser street detail, better enemy roles, one proper mission route
- Near-term milestone: a premium-feeling combat vertical slice
- Session focus: combat readability, district worldbuilding, mission flow

## Starfall Protocol

- Current state: orbital dock environment demo
- Immediate target: traversal prototype, scan/breach interactions, stealth-friendly layout
- Near-term milestone: one third-person-style mission slice in a contained zone
- Session focus: movement, interaction, level design, visual language

## Turbo Vector

- Current state: single-track marina racing demo
- Immediate target: AI rivals, garage/shop loop, one championship arc
- Near-term milestone: a proper `Super Cars`-style top-down racer prototype
- Session focus: handling, rivals, economy, track identity

## Redline Horizon

- Current state: single-route coastal road demo
- Immediate target: route splits, music-select framing, stage progression, richer traffic and scenery
- Near-term milestone: a `Lotus Esprit Turbo Challenge` / `OutRun` style arcade drive
- Session focus: road feel, atmosphere, progression, soundtrack identity
