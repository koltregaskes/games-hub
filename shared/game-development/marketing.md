# Marketing

## What good looks like

- The game has a clear one-sentence hook.
- The project produces repeatable content angles instead of random posts.
- The store page, demo, and public messaging match the real fantasy of the game.
- Marketing starts while the game is still being shaped, not a week before launch.

## Minimum studio-standard version

- One hook sentence.
- One target audience statement.
- Three content angles.
- One call to action for the current stage.

## Common mistakes

- Waiting until launch week to think about discoverability.
- Talking about features instead of fantasy.
- Posting screenshots with no context or public hook.
- Building a Steam page before the game has a sharp identity.

## Practical checklist

- Write the hook in plain English.
- Define the audience and why they would care.
- Decide what a trailer, GIF, or screenshot must communicate.
- Match your call to action to the stage: follow, wishlist, playtest, or buy.
- Review messaging when the game fantasy sharpens.
